Quelques définitions

HTML
L'HyperText Markup Language, généralement abréré HTML, est le format de données conçu pour représenter les pages Web
CSS
Les feuilles de styles en cascadel, énéralement apelées CSS de l'anglais Cascading styles Sheets, forment un langage informatique qui décrit la présentation des documents HTML et XML.
PHP
PHP : Hypertext Preprocessor, plus connu sous le sigle PHP (acronyme récursif), est un langage de programmation libre principalement utilisé pour produire des pages Web dynamiques via un serveur HTTP
